## 🏛️ 赛博联邦权力架构（复刻美国联邦政府体系）

四、核心功能与使用指南
1. 任务管理
创建任务：输入需求，选择优先级，提交后自动进入状态机流转

查看任务：列表页展示所有任务，支持状态筛选

任务详情：查看完整执行轨迹、流程图、审计日志、Token消耗

操作：暂停、终止、重试失败任务

2. Agent管理
查看所有Agent的状态（启用/禁用）、执行次数、Token消耗

启用/禁用Agent（需相应权限）

3. 插件管理
查看已加载插件列表，支持启用/禁用、重载、查看调用统计

插件权限等级与角色绑定，高等级插件仅限高权限角色调用

4. 审计日志
按任务ID、时间范围查询操作日志、执行日志、合规日志、Token日志

支持日志导出为JSON文件

5. 监控大屏
实时展示任务总数、成功率、运行中任务、Token消耗

任务状态分布图、Token消耗趋势、Agent调用次数、系统资源使用率

每30秒自动刷新

五、API文档
启动服务后访问 http://localhost:8000/docs 查看Swagger UI，包含以下主要接口：

任务管理：创建、查询、暂停、终止、重试

Agent管理：列表、详情、启用/禁用

插件管理：列表、启用/禁用、重载、统计

审计日志：查询、导出

系统状态：健康检查、预算查询、权限查询

六、生产部署注意事项
API密钥：务必修改 .env 中的 API_SECRET_KEY 为强密码

大模型密钥：如需使用真实模型，请配置对应API密钥；若无，系统将使用模拟模式

数据持久化：SQLite数据、日志、插件文件均已挂载卷，重启不丢失

高可用：可启动多个后端副本，依赖Redis分布式锁实现任务不重复执行

监控：已内置Prometheus指标端点 /metrics，可对接Grafana
